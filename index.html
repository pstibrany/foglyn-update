Update site

This is Update site for Foglyn. Update sites are used by Eclipse to install additional plugins. Please see Installation instructions for details about installing Foglyn.

Full URL of Update Site is: http://update.foglyn.com/stable

ZIP Archive is available too: foglyn-1.7.zip. Simply extract this ZIP archive into dropins folder of your Eclipse installation to install or update Foglyn. (Don't forget to use -clean parameter when you restart Eclipse next time).

See releases for more details about changes between releases.
